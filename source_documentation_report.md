## Niveaux de Fiabilité

### Élevée (🔒)
Données confirmées par source officielle ou plusieurs sources indépendantes. Exemples : sites officiels, communiqués de presse officiels, registres publics.

### Moyenne (🔓)
Données provenant de sources professionnelles ou spécialisées. Exemples : presse spécialisée, annuaires professionnels, biographies.

### Faible (⚠️)
Données partielles ou non vérifiées. Exemples : informations indirectes, sources non confirmées.

1. **Mise à jour régulière:** Les données sur les membres du bureau devraient être actualisées annuellement, voire semestriellement pour les associations majeures.
